Projetos técnicos

Uma seleção de projetos em Machine Learning, desenvolvimento web, mobile, desktop e jogos, mostrando minha base técnica e variedade de tecnologias.

Modelo de Machine Learning para Doenças Crônicas

Comparação de diferentes algoritmos (LightGBM, CatBoost, XGBoost, etc.) para predição de doenças crônicas em idosos, avaliando acurácia, precisão, recall, F1-score e AUC-ROC.

Tecnologias: Python, Pandas, scikit-learn, LightGBM.

Jogo da Memória (Web)

Jogo da memória com interface simples e visual em pixel art, desenvolvido para treinar atenção e lógica, com sistema de pontos e diferentes modos de dificuldade.

Tecnologias: HTML, CSS, JavaScript.

QuickList – App de lista de compras

Aplicativo mobile para organizar compras de forma rápida, com interface limpa, ícones ilustrativos e autenticação de usuário.

Tecnologias: Flutter, Dart.

Programa Reordenador de Texto

Aplicação desktop que manipula blocos de texto, permitindo reorganização e processamento de grandes quantidades de conteúdo, com diferentes temas visuais.

Tecnologias: Java (Swing/JavaFX ou equivalente).

Sistema de Arquivos Desktop

Ferramenta desktop para navegação, criação, exclusão e manipulação de arquivos em um ambiente gráfico, simulando um gerenciador de arquivos completo.

Tecnologias: C++ e Qt.

Exam Scape – Jogo de Terror Educacional

Jogo em primeira pessoa com atmosfera de terror, puzzles, cofre com combinação e interação com cenário, unindo experiência imersiva com temática educacional.

Tecnologias: Unity, C#.
